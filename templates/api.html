¿Qué es esta API?

La API del parqueadero proporciona acceso a la base de datos de estudiantes registrados que hacen uso del parqueadero de la Universidad Santo Tomás de Tunja. Permite realizar operaciones como agregar, actualizar, eliminar y recuperar información de los estudiantes.
